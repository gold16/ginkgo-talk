Quick Start

Up and Running in 30 Seconds

One binary, no installer. Works on Windows.

Terminal
# Option 1: Download pre-built binary from GitHub Releases
# Option 2: Build from source

$ git clone https://github.com/gold16/ginkgo-talk.git
$ cd ginkgo-talk
$ go build -ldflags "-H windowsgui" -o GinkgoTalk.exe .
$ ./GinkgoTalk.exe

# Open https://<LAN-IP>:9527/qrcode to see QR code
# Scan with your phone ‚Üí Enter pair code ‚Üí Done!
